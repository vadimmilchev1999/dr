# Happy Birthday — исправленный офлайн билд
Фиксы:
- Надёжный старт на мобильных: кнопка работает, тач по оверлею тоже; запуск разрешён только в портретной ориентации.
- Heart переделан на совместимый CSS (без clip-path:path) — работает в Safari/старых Android.
- Обновлённая последовательность: countdown → фраза (появление/исчезновение) → сердце → звёздное небо → книга.
- Музыка — только по иконке (playsinline).
Положите музыку в `music/happy-birthday.mp3`. Фотографии для книги меняются в массиве `PAGES` в `script.js`.
