# Happy Birthday — фиксированные файлы (портрет + обратный отсчёт + фраза + сердце + звёзды + книга)
- Старт только в вертикали. Если нажали «Начать» в горизонтали, сайт запустится автоматически после поворота.
- Отсчёт перезапускается корректно (точка падает каждый раз).
- Нет лишних кнопок — только иконка музыки (вверху справа).
- После фразы — сердце, затем звёздное небо и книга. Тап по книге листает страницы.

Музыка: положите `music/happy-birthday.mp3` рядом (папка `music/`).  
Фото: поменяйте массив `PAGES` в `script.js` на свои изображения.
