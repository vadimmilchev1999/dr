# Happy Birthday — клон «1 в 1» (горизонталь, точные тайминги)
Порядок: отсчёт → фраза (появление по буквам) → исчезновение фразы (взрыв наружу) → сердце (пульс+плавный fade-out) → звёздное небо → книга.
Ориентация: **только горизонталь** (если нажать «Начать» в вертикали, запуск произойдёт после поворота).
Музыка: файл `music/happy-birthday.mp3` рядом с `index.html` (папка `music/`). Управление — кнопка вверху справа.

Книга: сейчас заглушки (SVG). Замените в `script.js` массив `PAGES` на свои фото и текст.
